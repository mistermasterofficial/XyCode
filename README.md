# XyCode
 
Простой компилятор кода в инструкции для XyBeeDOS.

# Инструкции
Все инструкции начинаются с названия инструкции и параметров для неё, обязательно заканчиваются на ;.
- // КОММЕНТАРИЙ - комментарий, игнорируется при компиляции (требует ; в конце).
- include "path" - импортировать код из другого файла. Также можно импортировать файл из стандартной библиотеки ("STDLIB").
- tmpl НАЗВАНИЕ [ПЕРЕМЕННЫЕ_ДЛЯ_ПОДСТАНОВКИ] - шаблон кода для повторного использования набора инструкций через ". НАЗВАНИЕ". Обязательно закрыть инструкцией "endtmpl".
- func НАЗВАНИЕ [ЛОКАЛЬНЫЕ_ПЕРЕМЕННЫЕ] - определение функции с локальными переменными. Также можно добавлять переменные через "var [ЛОКАЛЬНЫЕ_ПЕРЕМЕННЫЕ]" к уже объявленным (повторы игнорируются). Обязательно закрыть инструкцией "endfunc".
- call НАЗВАНИЕ - вызов функции.
- if и endif - блок с условием.
- while и endwhile - цикл while.
- do и dowhile - цикл do-while.
